JAVAеҮ йҒ“еёёи§Ғзҡ„йқўиҜ•йўҳ

JavaиҮӘеёҰе“ӘеҮ з§ҚзәҝзЁӢжұ пјҹ

  1. newCachedThreadPool
    еҲӣе»әдёҖдёӘеҸҜзј“еӯҳзәҝзЁӢжұ пјҢеҰӮжһңзәҝзЁӢжұ й•ҝеәҰи¶…иҝҮеӨ„зҗҶйңҖиҰҒпјҢеҸҜзҒөжҙ»еӣһж”¶з©әй—ІзәҝзЁӢпјҢиӢҘж— еҸҜеӣһж”¶пјҢеҲҷж–°е»әзәҝзЁӢгҖӮиҝҷз§Қзұ»еһӢзҡ„зәҝзЁӢжұ зү№зӮ№жҳҜпјҡ
    е·ҘдҪңзәҝзЁӢзҡ„еҲӣе»әж•°йҮҸеҮ д№ҺжІЎжңүйҷҗеҲ¶пјҲе…¶е®һд№ҹжңүйҷҗеҲ¶зҡ„пјҢж•°зӣ®дёәInterger. MAX_VALUEпјү, иҝҷж ·еҸҜзҒөжҙ»зҡ„еҫҖзәҝзЁӢжұ дёӯж·»еҠ зәҝзЁӢгҖӮ
    еҰӮжһңй•ҝж—¶й—ҙжІЎжңүеҫҖзәҝзЁӢжұ дёӯжҸҗдәӨд»»еҠЎпјҢеҚіеҰӮжһңе·ҘдҪңзәҝзЁӢз©әй—ІдәҶжҢҮе®ҡзҡ„ж—¶й—ҙпјҲй»ҳи®Өдёә1еҲҶй’ҹпјүпјҢеҲҷиҜҘе·ҘдҪңзәҝзЁӢе°ҶиҮӘеҠЁз»ҲжӯўгҖӮз»ҲжӯўеҗҺпјҢеҰӮжһңдҪ еҸҲжҸҗдәӨдәҶж–°зҡ„д»»еҠЎпјҢеҲҷзәҝзЁӢжұ йҮҚж–°еҲӣе»әдёҖдёӘе·ҘдҪңзәҝзЁӢгҖӮ
    еңЁдҪҝз”ЁCachedThreadPoolж—¶пјҢдёҖе®ҡиҰҒжіЁж„ҸжҺ§еҲ¶д»»еҠЎзҡ„ж•°йҮҸпјҢеҗҰеҲҷпјҢз”ұдәҺеӨ§йҮҸзәҝзЁӢеҗҢж—¶иҝҗиЎҢпјҢеҫҲжңүдјҡйҖ жҲҗзі»з»ҹзҳ«з—ӘгҖӮ
  2. newFixedThreadPool
    еҲӣе»әдёҖдёӘжҢҮе®ҡе·ҘдҪңзәҝзЁӢж•°йҮҸзҡ„зәҝзЁӢжұ гҖӮжҜҸеҪ“жҸҗдәӨдёҖдёӘд»»еҠЎе°ұеҲӣе»әдёҖдёӘе·ҘдҪңзәҝзЁӢпјҢеҰӮжһңе·ҘдҪңзәҝзЁӢж•°йҮҸиҫҫеҲ°зәҝзЁӢжұ еҲқе§Ӣзҡ„жңҖеӨ§ж•°пјҢеҲҷе°ҶжҸҗдәӨзҡ„д»»еҠЎеӯҳе…ҘеҲ°жұ йҳҹеҲ—дёӯгҖӮFixedThreadPoolжҳҜдёҖдёӘе…ёеһӢдё”дјҳз§Җзҡ„зәҝзЁӢжұ пјҢе®ғе…·жңүзәҝзЁӢжұ жҸҗй«ҳзЁӢеәҸж•ҲзҺҮе’ҢиҠӮзңҒеҲӣе»әзәҝзЁӢж—¶жүҖиҖ—зҡ„ејҖй”Җзҡ„дјҳзӮ№гҖӮдҪҶжҳҜпјҢеңЁзәҝзЁӢжұ з©әй—Іж—¶пјҢеҚізәҝзЁӢжұ дёӯжІЎжңүеҸҜиҝҗиЎҢд»»еҠЎж—¶пјҢе®ғдёҚдјҡйҮҠж”ҫе·ҘдҪңзәҝзЁӢпјҢиҝҳдјҡеҚ з”ЁдёҖе®ҡзҡ„зі»з»ҹиө„жәҗгҖӮ
  3. newSingleThreadExecutor
    еҲӣе»әдёҖдёӘеҚ•зәҝзЁӢеҢ–зҡ„ExecutorпјҢеҚіеҸӘеҲӣе»әе”ҜдёҖзҡ„е·ҘдҪңиҖ…зәҝзЁӢжқҘжү§иЎҢд»»еҠЎпјҢе®ғеҸӘдјҡз”Ёе”ҜдёҖзҡ„е·ҘдҪңзәҝзЁӢжқҘжү§иЎҢд»»еҠЎпјҢдҝқиҜҒжүҖжңүд»»еҠЎжҢүз…§жҢҮе®ҡйЎәеәҸпјҲFIFO, LIFO, дјҳе…Ҳзә§пјүжү§иЎҢгҖӮеҰӮжһңиҝҷдёӘзәҝзЁӢејӮеёёз»“жқҹпјҢдјҡжңүеҸҰдёҖдёӘеҸ–д»Је®ғпјҢдҝқиҜҒйЎәеәҸжү§иЎҢгҖӮеҚ•е·ҘдҪңзәҝзЁӢжңҖеӨ§зҡ„зү№зӮ№жҳҜеҸҜдҝқиҜҒйЎәеәҸең°жү§иЎҢеҗ„дёӘд»»еҠЎпјҢе№¶дё”еңЁд»»ж„Ҹз»ҷе®ҡзҡ„ж—¶й—ҙдёҚдјҡжңүеӨҡдёӘзәҝзЁӢжҳҜжҙ»еҠЁзҡ„гҖӮ
  4. newScheduleThreadPool
    еҲӣе»әдёҖдёӘе®ҡй•ҝзҡ„зәҝзЁӢжұ пјҢиҖҢдё”ж”ҜжҢҒе®ҡж—¶зҡ„д»ҘеҸҠе‘ЁжңҹжҖ§зҡ„д»»еҠЎжү§иЎҢпјҢж”ҜжҢҒе®ҡж—¶еҸҠе‘ЁжңҹжҖ§д»»еҠЎжү§иЎҢгҖӮе»¶иҝҹ3з§’жү§иЎҢгҖӮ

HashMapе’ҢHashTableеҢәеҲ«

  1. зәҝзЁӢе®үе…ЁжҖ§дёҚеҗҢ
    HashMapжҳҜзәҝзЁӢдёҚе®үе…Ёзҡ„пјҢHashTableжҳҜзәҝзЁӢе®үе…Ёзҡ„пјҢе…¶дёӯзҡ„ж–№жі•жҳҜSynchronizeзҡ„пјҢеңЁеӨҡзәҝзЁӢе№¶еҸ‘зҡ„жғ…еҶөдёӢпјҢеҸҜд»ҘзӣҙжҺҘдҪҝз”ЁHashTablпјҢдҪҶжҳҜдҪҝз”ЁHashMapж—¶еҝ…йЎ»иҮӘе·ұеўһеҠ еҗҢжӯҘеӨ„зҗҶгҖӮ
  2. жҳҜеҗҰжҸҗдҫӣcontainsж–№жі•
    HashMapеҸӘжңүcontainsValueе’ҢcontainsKeyж–№жі•пјӣHashTableжңүcontainsгҖҒcontainsKeyе’ҢcontainsValueдёүдёӘж–№жі•пјҢе…¶дёӯcontainsе’ҢcontainsValueж–№жі•еҠҹиғҪзӣёеҗҢгҖӮ
  3. keyе’ҢvalueжҳҜеҗҰе…Ғи®ёnullеҖј
    HashtableдёӯпјҢkeyе’ҢvalueйғҪдёҚе…Ғи®ёеҮәзҺ°nullеҖјгҖӮHashMapдёӯпјҢnullеҸҜд»ҘдҪңдёәй”®пјҢиҝҷж ·зҡ„й”®еҸӘжңүдёҖдёӘпјӣеҸҜд»ҘжңүдёҖдёӘжҲ–еӨҡдёӘй”®жүҖеҜ№еә”зҡ„еҖјдёәnullгҖӮ
  4. ж•°з»„еҲқе§ӢеҢ–е’Ңжү©е®№жңәеҲ¶
    HashTableеңЁдёҚжҢҮе®ҡе®№йҮҸзҡ„жғ…еҶөдёӢзҡ„й»ҳи®Өе®№йҮҸдёә11пјҢиҖҢHashMapдёә16пјҢHashtableдёҚиҰҒжұӮеә•еұӮж•°з»„зҡ„е®№йҮҸдёҖе®ҡиҰҒдёә2зҡ„ж•ҙж•°ж¬Ўе№ӮпјҢиҖҢHashMapеҲҷиҰҒжұӮдёҖе®ҡдёә2зҡ„ж•ҙж•°ж¬Ўе№ӮгҖӮ
    Hashtableжү©е®№ж—¶пјҢе°Ҷе®№йҮҸеҸҳдёәеҺҹжқҘзҡ„2еҖҚеҠ 1пјҢиҖҢHashMapжү©е®№ж—¶пјҢе°Ҷе®№йҮҸеҸҳдёәеҺҹжқҘзҡ„2еҖҚгҖӮ

TreeSetе’ҢHashSetеҢәеҲ«

HashSetжҳҜйҮҮз”ЁhashиЎЁжқҘе®һзҺ°зҡ„гҖӮе…¶дёӯзҡ„е…ғзҙ жІЎжңүжҢүйЎәеәҸжҺ’еҲ—пјҢadd()гҖҒremove()д»ҘеҸҠcontains()зӯүж–№жі•йғҪжҳҜеӨҚжқӮеәҰдёәO(1)зҡ„ж–№жі•гҖӮ
TreeSetжҳҜйҮҮз”Ёж ‘з»“жһ„е®һзҺ°пјҲзәўй»‘ж ‘з®—жі•пјүгҖӮе…ғзҙ жҳҜжҢүйЎәеәҸиҝӣиЎҢжҺ’еҲ—пјҢдҪҶжҳҜadd()гҖҒremove()д»ҘеҸҠcontains()зӯүж–№жі•йғҪжҳҜеӨҚжқӮеәҰдёәO(log (n))зҡ„ж–№жі•гҖӮе®ғиҝҳжҸҗдҫӣдәҶдёҖдәӣж–№жі•жқҘеӨ„зҗҶжҺ’еәҸзҡ„setпјҢеҰӮfirst()пјҢlast()пјҢheadSet()пјҢtailSet()зӯүзӯүгҖӮ

String bufferе’ҢString buildеҢәеҲ«

  1. StringBufferдёҺStringBuilderдёӯзҡ„ж–№жі•е’ҢеҠҹиғҪе®Ңе…ЁжҳҜзӯүд»·зҡ„гҖӮ
  2. еҸӘжҳҜStringBufferдёӯзҡ„ж–№жі•еӨ§йғҪйҮҮз”ЁдәҶ synchronized е…ій”®еӯ—иҝӣиЎҢдҝ®йҘ°пјҢеӣ жӯӨжҳҜзәҝзЁӢе®үе…Ёзҡ„пјҢиҖҢStringBuilderжІЎжңүиҝҷдёӘдҝ®йҘ°пјҢеҸҜд»Ҙиў«и®ӨдёәжҳҜзәҝзЁӢдёҚе®үе…Ёзҡ„гҖӮ
  3. еңЁеҚ•зәҝзЁӢзЁӢеәҸдёӢпјҢStringBuilderж•ҲзҺҮжӣҙеҝ«пјҢеӣ дёәе®ғдёҚйңҖиҰҒеҠ й”ҒпјҢдёҚе…·еӨҮеӨҡзәҝзЁӢе®үе…ЁиҖҢStringBufferеҲҷжҜҸж¬ЎйғҪйңҖиҰҒеҲӨж–ӯй”ҒпјҢж•ҲзҺҮзӣёеҜ№жӣҙдҪҺ

FinalгҖҒFinallyгҖҒFinalize

finalпјҡдҝ®йҘ°з¬ҰпјҲе…ій”®еӯ—пјүжңүдёүз§Қз”Ёжі•пјҡдҝ®йҘ°зұ»гҖҒеҸҳйҮҸе’Ңж–№жі•гҖӮдҝ®йҘ°зұ»ж—¶пјҢж„Ҹе‘ізқҖе®ғдёҚиғҪеҶҚжҙҫз”ҹеҮәж–°зҡ„еӯҗзұ»пјҢеҚідёҚиғҪиў«з»§жүҝпјҢеӣ жӯӨе®ғе’ҢabstractжҳҜеҸҚд№үиҜҚгҖӮдҝ®йҘ°еҸҳйҮҸж—¶пјҢиҜҘеҸҳйҮҸдҪҝз”ЁдёӯдёҚиў«ж”№еҸҳпјҢеҝ…йЎ»еңЁеЈ°жҳҺж—¶з»ҷе®ҡеҲқеҖјпјҢеңЁеј•з”ЁдёӯеҸӘиғҪиҜ»еҸ–дёҚеҸҜдҝ®ж”№пјҢеҚідёәеёёйҮҸгҖӮдҝ®йҘ°ж–№жі•ж—¶пјҢд№ҹеҗҢж ·еҸӘиғҪдҪҝз”ЁпјҢдёҚиғҪеңЁеӯҗзұ»дёӯиў«йҮҚеҶҷгҖӮ
finallyпјҡйҖҡеёёж”ҫеңЁtryвҖҰcatchзҡ„еҗҺйқўжһ„йҖ жңҖз»Ҳжү§иЎҢд»Јз Ғеқ—пјҢиҝҷе°ұж„Ҹе‘ізқҖзЁӢеәҸж— и®әжӯЈеёёжү§иЎҢиҝҳжҳҜеҸ‘з”ҹејӮеёёпјҢиҝҷйҮҢзҡ„д»Јз ҒеҸӘиҰҒJVMдёҚе…ій—ӯйғҪиғҪжү§иЎҢпјҢеҸҜд»Ҙе°ҶйҮҠж”ҫеӨ–йғЁиө„жәҗзҡ„д»Јз ҒеҶҷеңЁfinallyеқ—дёӯгҖӮ
finalizeпјҡObjectзұ»дёӯе®ҡд№үзҡ„ж–№жі•пјҢJavaдёӯе…Ғи®ёдҪҝз”Ёfinalize() ж–№жі•еңЁеһғеңҫж”¶йӣҶеҷЁе°ҶеҜ№иұЎд»ҺеҶ…еӯҳдёӯжё…йҷӨеҮәеҺ»д№ӢеүҚеҒҡеҝ…иҰҒзҡ„жё…зҗҶе·ҘдҪңгҖӮиҝҷдёӘж–№жі•жҳҜз”ұеһғеңҫж”¶йӣҶеҷЁеңЁй”ҖжҜҒеҜ№иұЎж—¶и°ғз”Ёзҡ„пјҢйҖҡиҝҮйҮҚеҶҷfinalize() ж–№жі•еҸҜд»Ҙж•ҙзҗҶзі»з»ҹиө„жәҗжҲ–иҖ…жү§иЎҢе…¶д»–жё…зҗҶе·ҘдҪңгҖӮ
